# 课程资料智能问答助手

这是一个面向大学课程学习场景的 AI 应用项目。

## 项目目标

用户可以上传课程资料，例如 PDF、TXT、Markdown 文件，然后向系统提问。系统会先从课程资料中检索相关内容，再调用大模型生成回答，从而减少大模型胡编乱造的问题。
